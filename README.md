# YouTube clone React App
## Fetching YouTube video via YouTube API v3 with Search Helper Library.
## Tech Stacks
  + React
  + Lodash
  + Youtube-api-search
  + Bulma CSS
  + moment
  + font-awesome

```
git clone "https://github.com/yeondam88/react-youtube-clone.git"
To run locally.
yarn add
yarn start
To build
yarn build
```
